1xx – Informational
100 Continue → The server received part of the request and tells the client "continue sending"
101 Switching Protocols → The server agreed to switch the protocol (e.g., from HTTP to WebSocket)

2xx – Success
200 OK → The request succeeded and returned the requested data
201 Created → A new resource was successfully created (e.g., a message or conversation)
202 Accepted → The server accepted the request but has not processed it yet
204 No Content → The request succeeded but there is no content to return (e.g., successful deletion)

3xx – Redirection
301 Moved Permanently → The resource has permanently moved to a new location
302 Found → The resource is temporarily located at a different URL
304 Not Modified → The resource has not changed (useful for caching)

4xx – Client Error
400 Bad Request → The request is invalid (e.g., missing parameter or malformed input)
401 Unauthorized → The user is not authenticated (login required)
403 Forbidden → The user is authenticated but not allowed to perform this action
404 Not Found → The requested resource does not exist (e.g., conversation or message not found)
405 Method Not Allowed → The HTTP method (GET/POST/DELETE) is not allowed for this endpoint
409 Conflict → There is a conflict in the request (e.g., trying to create a resource that already exists)
422 Unprocessable Entity → The request data is understood but invalid (validation error)
429 Too Many Requests → The client sent too many requests in a short time (rate limit exceeded)

5xx – Server Error
500 Internal Server Error → A general server error occurred
501 Not Implemented → The server does not support the requested operation
502 Bad Gateway → The gateway server received an invalid response from an upstream server
503 Service Unavailable → The server is overloaded or under maintenance
504 Gateway Timeout → The gateway server did not receive a timely response from an upstream server
